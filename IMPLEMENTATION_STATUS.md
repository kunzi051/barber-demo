## 已知限制

1. 头发视觉使用 ColorRect 尺寸变化和颜色表示长度，非真实毛发效果
2. 对话面板点击快速连点可能有短暂闪烁
3. 理发师移动的碰撞检测范围较宽松（简单边界限制）
4. 窗口缩放到极端尺寸（<800×450）时 UI 可能重叠
